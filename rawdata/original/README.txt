These files contain data from Applied Regression Analysis,3rd Edition,
by Norman R. Draper and Harry Smith, published by John Wiley and Sons.

The file name indicates the source of the data, for example:
01A is Appendix A of Chapter 1
02.1 is Section 2.
16Ex1 is Example 1 of Chapter 16
23.3.3 is Equation (23.3.3)
e03a is Exercise A at the end of Chapter 3
e03aa is Exercise AA at the end of Chapter 3 
(Note that double letter exercises come out of order 
 in the list, but this occurs only in Chapter 3)

The complete listing of files is in the file list.data
